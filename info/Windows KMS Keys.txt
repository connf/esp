Guidance:
- Factory Reset via USB Recovery Download
- Unlicense Windows 10 Home via slmgr.vbs /upk (uninstall product key)
 && /cpky (clear product key from registry)
[ && /rilc re-install license files - Not required?]
[- Unlicense Windows 10 via MS Toolkit Settings > Show Unlicensed? NOT REQUIRED?]
- Check License with MS Toolkit
- - If Retail / Digital then Product Keys will not be accepted
- - - Install Product Key using System > Activation (Pro is accepted but unactivated until KMS runs)
- - - Reboot
- - - Active KMS Service and test activation
[- - - If fails AutoKMS and try again or AutoKMS first (then KMS Server / Service worked) - Not required?]
- - - System > Activation should now show Activated Win 10 Pro KMS / Organisation
{- - - Testing reboot to see if still activated now}
- - If Activated Win 10 Pro KMS / Organisation then KMS Keys should be acceptable and activate
- - - Install KMS Product Key for desired version (if not Pro and if accepted by system)
- - - Reboot to install OS differences (online required?)
- - - Test activation
- - - - Active KMS Service again if needed
